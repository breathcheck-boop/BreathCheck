# BreathCheck

BreathCheck is a local-first desktop app for anxiety self-help. It combines guided CBT-style learning, practical coping tools, and progress tracking.

## Download and Install

The download process is simple:
1. Click Download Files on the project webpage.
2. Download starts for the ZIP package.
3. Extract the files.
4. Open the app executable. It is ready to use after extraction.

## What the App Includes

### Main Navigation
- Learn
- Tools
- Progress
- Support
- Settings

### Learn Program (6 Modules)
1. Understanding Anxiety
2. Relaxation
3. Our Thoughts
4. Changing Thoughts
5. Coping with Worry
6. Lifestyle Factors

Each module is step-based, saves progress automatically, and unlocks in sequence.

### Tools
- BreathCheck breathing tool (pace, duration, session tracking)
- Thought Log (situation, thoughts, emotion rating, re-rate, saved history)

### Progress
- Program completion summary
- Weekly engagement summary
- Streak tracking
- Milestones

### Support
- Emergency support steps
- User-managed support contacts/resources

### Settings
- Change master password
- Reset progress
- Delete account / delete all data
- Comfort mode
- Font size

## Privacy and Safety

- Data is stored locally on your device.
- Sensitive text fields are encrypted before storage.
- BreathCheck is a self-help support app and not a replacement for emergency or professional mental health care.
