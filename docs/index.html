6-Module Program

Modules unlock sequentially to keep progression clear and structured.

1. Understanding Anxiety

Anxiety basics, body signals, worry cycle, values, baseline, and personal planning.

2. Relaxation

Body calming techniques, breathing routines, and tension-reduction practice.

3. Our Thoughts

Identify common anxious thought patterns and assumptions.

4. Changing Thoughts

Practice balanced thinking and cognitive reframing.

5. Coping with Worry

Separate productive problem-solving from repetitive worry loops.

6. Lifestyle Factors

Sleep, nutrition, movement, mindfulness, and relapse-prevention planning.

Privacy and Safety

Local-first by default

BreathCheck stores data on the user device and uses encrypted fields for sensitive entries.

Supportive, not emergency care

The app is designed for self-help and skills practice. Users should seek professional help when needed.
