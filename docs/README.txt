BreathCheck Showcase Webpage

Files:
- index.html
- styles.css
- script.js

How to open:
1) Open `showcase_web/index.html` directly in a browser, or
2) Run a local static server from project root:
   python -m http.server 8000
   then visit: http://localhost:8000/showcase_web/

Notes:
- The page references assets from `src/anxiety_app/ui/resources/...`.
- Design tokens follow the BreathCheck app palette (soft green, calm cards, rounded corners).
